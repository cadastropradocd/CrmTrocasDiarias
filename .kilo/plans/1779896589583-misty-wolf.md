# Plano de Migração: Next.js + Cloudflare Pages + D1

## Stack Target (Cloudflare Full-Stack)
- **Next.js 16** - Framework full-stack
- **Cloudflare Pages** - Hosting com `@opennextjs/cloudflare`
- **D1** - Banco SQLite serverless na edge
- **Wrangler** - CLI para deploy

## Estrutura Final
```
/ (raiz)
├── app/                    # Frontend Next.js (movido de /old/app/)
├── public/                 # Assets estáticos (movido de /old/public/)
├── schema.sql              # Schema D1
├── wrangler.toml           # Config Pages + D1
├── package.json            # Dependências
└── /old                    # Backup do código antigo
```

## Fluxo de Dados
```
Browser → Next.js (Cloudflare Pages) → D1 (SQLite edge)
```

---

## Etapas de Migração

### 1. Preparar Estrutura
- [ ] Mover `/old/app/` → `/app/`
- [ ] Mover `/old/public/` → `/public/`
- [ ] Atualizar imports que referenciam `@/app/` para `@/`
- [ ] Atualizar `package.json` com dependências necessárias

### 2. Configurar D1
- [ ] Criar D1: `wrangler d1 create trocas-diarias`
- [ ] Atualizar `wrangler.toml` com binding D1
- [ ] Aplicar schema: `wrangler d1 execute trocas-diarias --remote --file=./schema.sql`
- [ ] (Opcional) Seed: `wrangler d1 execute trocas-diarias --remote --file=./seed.sql`

### 3. Ajustar API Routes para D1
- [ ] Atualizar `/app/api/*` para usar D1 via `env.DB`
- [ ] Garantir que auth usa `env.JWT_SECRET`
- [ ] Manter cookies HttpOnly (já funciona)

### 4. Configurar Wrangler para Pages
- [ ] Criar/ajustar `wrangler.toml` para Pages
- [ ] Configurar build command: `npm run build`
- [ ] Configurar output directory: `.next`

### 5. Variáveis de Ambiente
- [ ] Configurar `JWT_SECRET` no Cloudflare Pages settings
- [ ] (Opcional) `NEXT_PUBLIC_*` se precisar

### 6. Deploy e Testes
- [ ] Testar local: `wrangler pages dev .next` ou `npm run dev`
- [ ] Deploy: `wrangler pages deploy .next`
- [ ] Verificar D1 no dashboard

---

## Comandos Principais

```bash
# 1. Criar D1
wrangler d1 create trocas-diarias

# 2. Aplicar schema (copiar database_id do passo anterior)
wrangler d1 execute trocas-diarias --remote --file=./schema.sql

# 3. Deploy
wrangler pages deploy .next

# 4. Configurar secrets no dashboard
# Cloudflare Pages → Settings → Environment Variables
```

## Decisões Tomadas
- **Frontend**: Next.js (movido de /old)
- **Backend**: API routes do Next.js (mesmo código)
- **Database**: D1 apenas (sem Supabase)
- **Deploy**: Cloudflare Pages
